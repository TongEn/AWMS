名词缩写及释义
==============

**Cor**：Chain of Responsibility，最高層物流知識節點

**VM**：Vistor Manager，運算管理員

**Visitor**：最小運算單元


**Docker**：应用容器引擎,轻量级的虚拟化技术

**K8S**: Kubernetes,管理云平台中多个主机上的容器化的应用

**Kafka**：訊息佇列 (message queue) 分散式運算

**Redis**：內存資料庫 In-memory database 快速查詢

**mongoDB**：長存資料庫 persistence database 長期儲存


CORList
    **SalesOrderCor**: 訂單服务
    
    **SalesOBndCor**: 出貨服务
    
    **InventoryCor**: 庫存服务
    
    **SupplementShiftCor**: 截单補貨服务
